# Robot cleaner

[![Build Status](https://travis-ci.org/dim2k2006/robot-cleaner.svg?branch=master)](https://travis-ci.org/dim2k2006/robot-cleaner)

## Setup

```sh
$ make install
```

## Start

```sh
$ make start
```

## Publish

```sh
$ make publish
```
